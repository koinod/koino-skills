# /cold-outreach

Generate a personalized 5-touch email sequence + 3 LinkedIn DM variants for any prospect using proven sales frameworks.

## Usage

```
/cold-outreach
Prospect: Sarah Chen
Company: CloudMetrics
Industry: SaaS / Analytics
Pain point: 40% meeting no-show rate
My offer: AI meeting confirmation system
```

## What You Get

**5 Emails** (each using a different framework):
1. The Opener (PAS framework) — Day 1
2. The Value Drop (Before-After-Bridge) — Day 3
3. The Social Proof (Case Study) — Day 7
4. The Pattern Interrupt (Curiosity + Contrast) — Day 12
5. The Breakup (Loss Aversion) — Day 18

**3 LinkedIn DMs**:
1. Connection request note (300 chars)
2. Follow-up value drop (after accept)
3. Pivot to conversation (Day 5-7)

**Plus**: Full timing strategy, objection handling scripts, and batch mode for multiple prospects.
