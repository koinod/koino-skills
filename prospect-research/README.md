# /prospect-research

Research any company and generate a structured intelligence profile — industry, team size, tech stack, recent news, pain points, and strategic outreach angles.

## Usage

```
/prospect-research Gong.io
/prospect-research "Apex HVAC Services"
```

Or provide more context:

```
/prospect-research
Company: Stripe
Contact: Patrick Collison
Context: We sell developer tools
```

## What You Get

- Company overview (industry, size, funding, business model)
- Products and services breakdown
- Tech stack (from job postings, integrations, BuiltWith)
- Key people with relevant context
- Recent news and activity (last 90 days)
- Strategic outreach angles (pain points, conversation starters, timing signals)
- Competitive landscape
- Pre-call checklist

## Batch Mode

Pass multiple companies to get a profile for each:

```
/prospect-research
1. Gong.io
2. Outreach.io
3. Apollo.io
```
