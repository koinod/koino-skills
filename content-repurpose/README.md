# /content-repurpose

Turn one piece of long-form content into 10 platform-optimized social media assets — each with a hook, body, and CTA.

## Usage

```
/content-repurpose
```

Then paste your content (blog post, transcript, article, newsletter).

## What You Get

| # | Asset | Platform |
|---|-------|----------|
| 1-3 | LinkedIn posts (contrarian, story, data angles) | LinkedIn |
| 4-6 | Tweets (hot take, tactical, micro-story) | Twitter/X |
| 7-8 | IG captions with visual suggestions + hashtags | Instagram |
| 9 | Email newsletter with subject line + preview text | Email |
| 10 | Carousel outline (8 slides) | LinkedIn/IG |

Plus a suggested posting schedule for the week.

## Supported Input

- Blog posts
- Podcast/video transcripts
- Newsletter issues
- Articles or essays
- Sales call recordings (transcribed)
- Webinar content

## Install

Add to your Claude Code skills directory or invoke directly.
