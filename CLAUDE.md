# CLAUDE.md

Guidance for Claude Code when working in this repository.

## Mission

Public-facing trim subset of agent skills for distribution. 14 polished skills, freemium positioning. The full operational 76-skill master is at koinod/skills.

## Stack

Pure markdown SKILL.md files + optional configs.

## First-time orientation

Before doing substantive work in this repo, read in order:

1. **[ceoian/agent-learnings/CLAUDE.md](https://github.com/ceoian/agent-learnings/blob/main/CLAUDE.md)** — org-wide operating principles for any Claude session.
2. **[ceoian/agent-learnings/INDEX.md](https://github.com/ceoian/agent-learnings/blob/main/INDEX.md)** — failure log + patterns to avoid repeating mistakes.
3. **This repo's `README.md`** — what it is and how to run it.

## Principles

- Keep skills here LITE — sufficient to demo value, not the full alpha. Full versions live at koinod/skills.
- Each skill needs SKILL.md frontmatter (name + description) so it's discoverable.

## Don'ts

- Don't add internal-only or alpha skills here — those belong at koinod/skills only.
